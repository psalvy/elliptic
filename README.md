# Elliptic Curves & Ramanujan's π

An interactive web course connecting the geometry of tori, the algebra of elliptic curves, and Ramanujan's extraordinary formulas for π.

## Who it's for

STEM learners comfortable with complex numbers, Euler's formula, and the basics of Riemann surfaces — but who want visual intuition for quotient spaces and the deeper structure behind Ramanujan's series.

## Modules

| # | Title | Topics |
|---|-------|--------|
| 1 | Demystifying Quotient Spaces | ℝ/ℤ → circle, ℂ/Λ → torus, gluing animation |
| 2 | Elliptic Curves over ℝ | Weierstrass form, group law, point addition |
| 3 | Elliptic Curves over ℂ | Weierstrass ℘-function, lattice ↔ curve correspondence |
| 4 | The j-invariant & Modular Forms | Modular group, upper half-plane, fundamental domain |
| 5 | Complex Multiplication & Ramanujan's π | Singular moduli, CM lattices, series convergence |
| 6 | Deriving the Formula | Legendre's relation, periods, quasi-periods |

## How to run

No build step required — open `index.html` in a browser.

```
open index.html
```

All dependencies (Three.js, KaTeX) are bundled under `lib/`.
