Elliptic Curves over ℝ

2.1 — The Weierstrass Form

An elliptic curve over the reals is defined by:

$$y^2 = x^3 + ax + b$$

The curve is non-singular (no cusps or self-intersections) exactly when the discriminant is nonzero:

$$\Delta = -16(4a^3 + 27b^2) \neq 0$$

2.2 — The Group Law: Point Addition

Elliptic curves have a remarkable group structure. Given two points $P$ and $Q$ on the curve:

  1. Draw the line through $P$ and $Q$. It intersects the curve at a third point $R$.
  2. Reflect $R$ over the $x$-axis to get $P + Q$.

When $P = Q$, use the tangent line instead (point doubling).

2.3 — Special Cases

Try these parameter values to see important curve shapes:

  • $a = -1, b = 0$ — curve passes through the origin, has two components
  • $a = -3, b = 2$ — singular! ($\Delta = 0$), the curve has a node
  • $a = 0, b = 0$ — singular cusp at the origin
  • $a = -1, b = 1$ — a nice smooth curve with one component
Key insight: The set of points on a non-singular elliptic curve, together with a "point at infinity" $\mathcal{O}$, forms an abelian group. This algebraic structure is what makes elliptic curves so powerful in number theory and cryptography.
