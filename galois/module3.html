The Galois Correspondence

3.1 — Fixed Fields

Let $K/F$ be a Galois extension with Galois group $G = \text{Gal}(K/F)$. For any subgroup $H \leq G$, define the fixed field:

$$K^H = \{\alpha \in K : \sigma(\alpha) = \alpha \text{ for all } \sigma \in H\}$$

This is always an intermediate field $F \subseteq K^H \subseteq K$. The bigger $H$ is, the more elements it fixes — or rather, the fewer elements survive all those fixings — so $K^H$ is smaller when $H$ is larger. This order-reversal is the heart of the correspondence.

3.2 — The Fundamental Theorem of Galois Theory

For a finite Galois extension $K/F$ with group $G$, there is a perfect bijection:

$$\left\{\begin{array}{c}\text{subgroups } H \leq G\end{array}\right\} \xleftrightarrow{1:1} \left\{\begin{array}{c}\text{intermediate fields} \\ F \subseteq M \subseteq K\end{array}\right\}$$

given by:

  • $H \mapsto K^H$ (fixed field of $H$)
  • $M \mapsto \text{Gal}(K/M)$ (the subgroup fixing $M$)

These maps are inverse to each other: $(K^H)^{\text{Gal}} = H$ and $K^{\text{Gal}(K/M)} = M$. The correspondence reverses inclusion: $H_1 \leq H_2 \Leftrightarrow K^{H_1} \supseteq K^{H_2}$.

Degree formulas: $[K : K^H] = |H|$ and $[K^H : F] = [G : H]$ (the index of $H$ in $G$).

Normality: A subgroup $H \trianglelefteq G$ is normal if and only if the corresponding fixed field $K^H/F$ is itself a Galois extension. In that case $\text{Gal}(K^H/F) \cong G/H$.

3.3 — Worked Example: Splitting Field of $x^3 - 2$

We work out the full correspondence for $K = \mathbb{Q}(\sqrt[3]{2}, \omega)$ over $F = \mathbb{Q}$, with $G = S_3$. The subgroups of $S_3 = \{e, (12), (13), (23), (123), (132)\}$:

The normal subgroup $A_3 \trianglelefteq S_3$ corresponds to the Galois intermediate extension $\mathbb{Q}(\omega)/\mathbb{Q}$, with $\text{Gal}(\mathbb{Q}(\omega)/\mathbb{Q}) \cong S_3/A_3 \cong \mathbb{Z}/2\mathbb{Z}$ (complex conjugation on $\omega$). The three non-normal subgroups correspond to the three non-Galois subextensions $\mathbb{Q}(r_i)$ — each contains only one real cube root and is not stable under the other automorphisms.

Deep Dive: Why Does $A_3$ Fix $\mathbb{Q}(\omega)$?

We need to check that the three-cycles $(123)$ and $(132)$ fix $\omega$. The three-cycle $(123)$ sends $r_1 \to r_2 \to r_3 \to r_1$. Since $r_2 = \omega r_1$ and $r_3 = \omega^2 r_1$, the automorphism $(123)$ must send $r_1 \mapsto \omega r_1$. For this to be a field automorphism fixing $\mathbb{Q}$, we need $\sigma(\omega r_1) = \omega \sigma(r_1) = \omega r_2 = \omega^2 r_1$. So $\sigma(r_1) = r_2$ and $\sigma(\omega) = \omega$ (since $\sigma(\omega r_1) = \sigma(\omega)\sigma(r_1) = \sigma(\omega) r_2 = \omega r_2 = \omega^2 r_1$, which requires $\sigma(\omega) = \omega$). Therefore $A_3$ fixes $\omega$, confirming $K^{A_3} \supseteq \mathbb{Q}(\omega)$. Since $[K : \mathbb{Q}(\omega)] = 3 = |A_3|$, we get equality.
